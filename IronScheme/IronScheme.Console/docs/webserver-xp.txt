Completing the websample installation
=====================================

- Copy all the dll files from the IronScheme installation directory to the 'bin' directory. 
- Copy the 'ironscheme' directory from the IronScheme installation directory to the 'bin' directory.
- Optionally copy the 'srfi' & 'lib' directories from the IronScheme installation directory to the 'bin' directory, if needed. 

or 

Run the setup-xp.bat

or

Use some other symlinking tool.
